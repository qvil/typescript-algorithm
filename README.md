# Typescript Algorithm

## Prerequisites

- Node.js >= 12.x (Recommended the latest version)
- (Yarn(Recommended))

## Install

```sh
yarn
# or
npm install
```

## Development

```sh
yarn dev
# or
npm run dev
```

## Build

```sh
yarn build
# or
npm run build
```

## Start

```sh
yarn start
# or
npm start
```
